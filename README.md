# Altın Meşe - Lüks Mobilya Web Sitesi

Modern ve zarif bir lüks mobilya web sitesi. Altın Meşe markası için özel olarak tasarlanmış, responsive ve kullanıcı dostu bir web deneyimi.

## Dosya Yapısı

```
altin_mese/
├── index.html      # Ana HTML dosyası
├── styles.css      # CSS stilleri ve animasyonlar
├── script.js       # JavaScript fonksiyonları
└── README.md       # Proje dokümantasyonu
```

## Özellikler

- ✅ **Responsive Tasarım**: Tüm cihazlarda mükemmel görünüm (mobil, tablet, masaüstü)
- ✅ **Modern Animasyonlar**: Smooth scroll ve fade-in animasyonları
- ✅ **Lüks Tasarım**: Altın ve kahverengi tonlarında zarif renk paleti
- ✅ **Hamburger Menü**: Mobil cihazlar için optimize edilmiş navigasyon
- ✅ **İletişim Formu**: Validasyonlu iletişim formu
- ✅ **Smooth Scroll**: Yumuşak sayfa içi geçişler
- ✅ **Özel Fontlar**: Cormorant (başlıklar) ve Inter (gövde metni)
- ✅ **Özel Scrollbar**: Tasarıma uygun özelleştirilmiş scrollbar

## Bölümler

1. **Hero Section**: Etkileyici giriş bölümü
2. **Tasarımlar**: Ürün tasarımları grid görünümü
3. **Hikâyemiz**: Marka hikayesi ve istatistikler
4. **Ustalık**: El işçiliği özellikleri
5. **Müşteri Yorumu**: Testimonial bölümü
6. **İletişim**: İletişim bilgileri ve randevu formu
7. **Footer**: Sosyal medya linkleri ve telif hakkı

## Kullanım

1. Tüm dosyaların aynı klasörde olduğundan emin olun
2. `index.html` dosyasını bir web tarayıcısında açın
3. Harici bağımlılık yoktur - sadece Google Fonts kullanılmaktadır

## Renk Paleti

- **Primary Background**: `#5C4A3A` (Koyu kahverengi)
- **Secondary Background**: `#6B5644` (Orta kahverengi)
- **Light Background**: `#F5EFE6` (Açık krem)
- **Gold Accent**: `#B8904D` (Altın)
- **Text Light**: `#E8D5C4` (Açık krem metin)
- **Text White**: `#FFFAF0` (Beyaz metin)

## Özelleştirme

### Renkleri Değiştirme
`styles.css` dosyasındaki `:root` değişkenlerini düzenleyerek renkleri özelleştirebilirsiniz:

```css
:root {
    --primary-bg: #5C4A3A;
    --gold: #B8904D;
    /* ... diğer renkler */
}
```

### İçerik Düzenleme
`index.html` dosyasındaki metinleri ve bölümleri düzenleyebilirsiniz.

### Form Gönderimi
`script.js` dosyasındaki form submit handler'ında, gerçek bir backend entegrasyonu ekleyebilirsiniz.
